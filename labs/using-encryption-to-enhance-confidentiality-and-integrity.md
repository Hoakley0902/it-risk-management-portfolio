# Using Encryption to Enhance Confidentiality and Integrity

## Overview
This project focused on implementing encryption techniques to protect data confidentiality and integrity. The work explored both symmetric and asymmetric encryption methods, digital signatures, and hybrid encryption models used in real-world systems. It demonstrated how encryption protects sensitive information during storage and transmission while ensuring that messages cannot be altered without detection. :contentReference[oaicite:1]{index=1}

## Skills Demonstrated
- Symmetric and asymmetric encryption
- Public and private key management
- Hybrid encryption implementation
- Digital signature creation
- Signature verification
- Secure file transfer
- Encryption key exchange
- Command-line cryptography tools

## Tools and Technologies Used
- OpenSSL
- GPG (GNU Privacy Guard)
- Kleopatra
- WinSCP
- Linux command line
- Public / Private key infrastructure (PKI)

## What I Worked On
In this project, I generated asymmetric key pairs and exchanged public keys between users to establish secure communication. I encrypted and decrypted files using both symmetric and asymmetric methods, then examined the resulting ciphertext and decrypted output to confirm data integrity. The process included creating encrypted message files and verifying that only the intended recipient could successfully decrypt them. :contentReference[oaicite:2]{index=2}

I also demonstrated hybrid encryption by encrypting a symmetric key with a public key before transmitting it securely between systems. After transferring encrypted files, I decrypted the key and message to confirm that the data remained intact. In the challenge section, I digitally signed a document using GPG and verified the signature with Kleopatra to confirm authenticity and ensure that the message had not been altered during transmission. :contentReference[oaicite:3]{index=3}

## Key Takeaways
This project reinforced how encryption protects both the confidentiality and integrity of data. It also demonstrated why hybrid encryption models are widely used in modern security systems, combining the speed of symmetric encryption with the secure key exchange provided by asymmetric cryptography. From a practical IT and security perspective, the project highlighted how encryption, key management, and digital signatures work together to secure communications and validate trusted information sources.
