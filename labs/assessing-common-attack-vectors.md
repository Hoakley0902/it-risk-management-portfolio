# Assessing Common Attack Vectors

## Overview
This project focused on analyzing several common cyberattack techniques used to compromise systems and applications. The work explored how attackers exploit vulnerabilities through web injection attacks, malware payloads, denial-of-service activity, and social engineering techniques. The goal was to better understand how these attack vectors operate and how layered defensive strategies can mitigate their impact. :contentReference[oaicite:1]{index=1}

## What I Worked On
In this project, I explored multiple attack techniques used by threat actors to compromise systems. I performed injection attacks against vulnerable web applications, including both DOM-based and reflected cross-site scripting (XSS), demonstrating how malicious scripts can execute through manipulated user input. These exercises highlighted how improperly validated inputs can allow attackers to gain elevated access within web applications. :contentReference[oaicite:2]{index=2}

I also generated a malicious payload using `msfvenom` and simulated malware execution within a controlled environment to observe how attackers establish system access and gather system information. Additionally, I examined denial-of-service behavior by observing newly recruited hosts participating in a distributed attack and analyzing how service availability can be disrupted under heavy traffic conditions. Finally, I created a simulated phishing campaign using the Social Engineering Toolkit to demonstrate how attackers attempt to harvest credentials through deceptive email communication. :contentReference[oaicite:3]{index=3}

## Key Takeaways
This project reinforced how modern cyberattacks often combine multiple techniques to bypass security defenses. It also emphasized the importance of defensive strategies such as input validation, network monitoring, employee security awareness training, endpoint protection, and layered infrastructure defenses. From a security and IT operations perspective, understanding these attack vectors is essential for building effective detection and prevention mechanisms across enterprise systems.
