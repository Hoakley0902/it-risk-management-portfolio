# IT Risk Management Lab Portfolio

This repository contains hands-on labs completed through IT Risk Management coursework at the University of South Florida. It is organized as a professional portfolio to highlight technical skills relevant to IT support, web development, and entry-level technology roles.

## Skills Highlighted
- Risk assessment
- Vulnerability analysis
- Network traffic analysis
- Authentication and access control
- Security monitoring
- Backup and recovery
- Incident response
- Technical documentation

## Lab Index
- Lab 01 – Seven Domains of IT Infrastructure
- Lab 02 – Vulnerability Assessment and Scanning
- Lab 03 – Packet Capture and Traffic Analysis
- Lab 04 – User Authentication and Access Controls
- Lab 05 – Encryption, Confidentiality, and Integrity
- Lab 06 – Assessing Common Attack Vectors
- Lab 07 – Implementing IT Security Policy
- Lab 08 – Security Monitoring and Logging
- Lab 09 – Backup and Recovery
- Lab 10 – Incident Response and Forensics
