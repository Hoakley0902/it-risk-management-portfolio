# IT Risk Management Lab Portfolio

This repository contains hands-on labs completed through IT Risk Management coursework at the University of South Florida. It is organized as a professional portfolio to highlight technical skills relevant to IT support, web development, cybersecurity, and entry-level technology roles.

## Skills Highlighted
- Risk assessment
- Vulnerability analysis
- Network traffic analysis
- Authentication and access control
- Security monitoring and logging
- Encryption and data protection
- Backup and disaster recovery
- Incident response and digital forensics
- Technical documentation

## About

Each lab includes:

- A brief overview of the security concept
- Skills demonstrated during the lab
- Tools and technologies used

- Supporting documentation showing evidence of the completed lab

These projects demonstrate practical experience with core cybersecurity and IT infrastructure concepts including vulnerability assessment, monitoring, authentication controls, encryption, backup systems, and incident response investigation.
